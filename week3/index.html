CLAUDE.md 란?

프로젝트 폴더에 두는 규칙 파일입니다. 클로드 코드가 매 세션을 시작할 때 자동으로 읽습니다.

2주차 마지막에 잠깐 보았던 파일입니다. 이번 주에는 실제로 만들어서 써봅니다.

연구실 CLAUDE.md 예시

# 우리 연구실 규칙
## 논문 파일
- 파일명은 {저자}_{연도}_{제목30자}.pdf 형식으로 통일합니다.
- 새로 추가된 논문은 papers/inbox/에서 papers/reviewed/로 이동합니다.
## 인용 형식
- 기본은 Vancouver 스타일을 사용합니다.
- 학회 발표 자료에서는 APA 를 함께 지원합니다.
## 익명화
- 환자 실명은 P001, P002 … 형식으로 치환합니다.
- 생년월일은 연도만 남기고 월/일은 제거합니다.
## 응답 톤
- 한국어는 격식 있는 ~합니다 종결로 작성합니다.

vibe-coding-lab

새로 만들기 — /init

클로드 코드 안에서 /init 명령어를 입력하면, 현재 폴더를 스캔해서 CLAUDE.md 초안을 자동으로 만들어줍니다.

$ CLAUDE CODE
> /init
— 폴더를 분석해서 CLAUDE.md 초안을 만들어드리겠습니다 —

초안이 만들어지면 필요한 규칙을 직접 추가하거나 다듬어 완성합니다.

스킬이란?

특정 작업에 이름을 붙여 저장한 것입니다. PDF 다루기, 슬라이드 만들기, 엑셀 처리 등 반복되는 작업들이 각각 하나의 스킬로 정리되어 있습니다.

가장 좋은 점은 내가 만들지 않아도 된다는 것입니다. 이미 다른 사람들이 만들어둔 스킬을 가져다 씁니다.

13/29
vibe-coding-lab

Anthropic 공식 스킬 저장소

$ REPOSITORY
github.com/anthropics/skills
SCREENSHOT
github.com/anthropics/skills 저장소 첫 화면
skills/ 폴더 아래 여러 스킬 디렉토리가 보이도록 캡처

vibe-coding-lab

이런 스킬들이 있습니다

  • frontend-slides HTML 슬라이드 발표 자료 자동 생성
  • academic-pptx 학술 발표용 PowerPoint 자료 생성
  • pdf PDF 텍스트/표/이미지 추출, OCR
  • xlsx 엑셀 파일 읽기·쓰기·집계
  • docx Word 문서 작성·편집
  • … 이 밖에도 계속 늘어나고 있습니다

스킬 설치하기

원하는 스킬을 ~/.claude/skills/ 폴더에 두면 클로드 코드가 자동으로 인식합니다.

$ TERMINAL
$ cd ~/.claude/skills
$ git clone https://github.com/anthropics/skills.git anthropic-skills
— 이제 클로드 코드에서 스킬을 사용할 수 있습니다 —

설치가 어려우면 클로드에게 부탁하셔도 됩니다. > anthropic 공식 스킬 저장소를 내 클로드 폴더에 설치해줘

시나리오

두 가지 선택지

OPTION A
Marp — 마크다운으로

frontend-slides 스킬이 대신 만듭니다. 인터랙션 · 차트 · 비디오가 가능합니다.
장점: 표현력 · 인터랙티브 · 이 워크샵 슬라이드가 이 방식입니다

Marp — 마크다운으로 만들기

슬라이드 사이는 --- 로 구분합니다. 텍스트만 적으면 발표 자료가 됩니다.

---
marp: true
---
# 첫 번째 슬라이드
본문 한 줄
---
# 두 번째 슬라이드
- 항목 1
- 항목 2
> 이 파일을 Marp 로 렌더링해서
slides.html 로 만들어줘
