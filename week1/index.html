사람과 AI의 역할

HUMAN · 사람이 결정합니다

판단과 책임

  • 목표를 정합니다.
  • 자료를 준비합니다.
  • 결과를 판단합니다.
  • 공개 전 내용을 검수합니다.
AI · 작업을 수행합니다

작성과 반복

  • 코드를 작성합니다.
  • 파일을 수정합니다.
  • 디자인을 조정합니다.
  • 반복 작업을 처리합니다.
!
주의할 점
AI가 항상 맞는 결과를 내지는 않습니다. 연구 내용과 개인정보는 사람이 검수해야 하며, 좋은 홈페이지인지 판단하는 책임은 사람에게 있습니다.

두 종류의 연구실 홈페이지

  • 최신 정보

바이브코딩 방식

User 졸업생은 Alumni로 옮기고,
새 학생 추가하고,
새 논문도 올려줘.
AI 관련 파일을 찾고 수정합니다. People, Publications, News처럼 바뀌어야 할 위치를 함께 확인합니다.
User 화면 확인하고 이상한 부분만 다시 고쳐줘.
AI 결과를 확인하고 다시 반영합니다. 사람은 최종 내용과 개인정보만 검수합니다.
