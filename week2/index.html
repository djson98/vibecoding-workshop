권한 — 파일·명령 실행 전에 물어봅니다

Claude Code는 코드 편집·터미널 명령 실행 전 항상 사용자 확인을 받습니다. 처음에 매번 묻는 건 정상이고, 안전을 위한 디자인입니다.
1. ALLOW ONCE
이번 한 번만 허용
2. ALLOW ALWAYS
앞으로 같은 작업 자동 허용
3. DENY
거부하고 다른 방법 제안받기
* 허용한 권한 관리·해제는 /permissions

CLAUDE.md — 프로젝트 메모 파일

프로젝트 루트에 두는 메모 파일. Claude이 매 세션 시작할 때 자동으로 읽습니다. 프로젝트 규칙·맥락·자주 쓰는 명령어를 적어두면 매번 설명 안 해도 됨.
무엇을 적나
  • 프로젝트 개요 / 폴더 구조
  • 코딩 컨벤션, 금기사항
  • 자주 쓰는 빌드·테스트 명령어
  • 주의할 함정 / 알아둘 맥락
어떻게 만드나
> /init
현재 프로젝트를 스캔해서 자동으로 초안을 생성합니다.
